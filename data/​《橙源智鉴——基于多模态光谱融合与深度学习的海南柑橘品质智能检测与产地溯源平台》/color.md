# 橙源智鉴 (Chengyuan Web) - 项目色彩规范 (Color Palette)

本项目主要采用**橙色 (Orange)** 作为主色调，搭配**生态绿 (Green)** 作为强调色，并辅以蓝色、红色等作为状态指示色。以下是项目中使用的具体颜色值（HEX 格式），方便用于设计和绘图。

## 1. 核心主题色 (Theme Colors)

| 颜色名称 | HEX 色值 | 用途说明 |
| :--- | :--- | :--- |
| **主色 (Primary)** | `#EA580C` | 品牌主色调，用于主要按钮、高亮文本、图表核心数据 (对应 Tailwind `orange-600`) |
| **背景色 (Background)** | `#FFF7ED` | 页面全局背景色，温暖的浅橙色 (对应 Tailwind `orange-50`) |
| **文本色 (Foreground)** | `#431407` | 全局主要文本颜色，深褐色，对比度高且柔和 (对应 Tailwind `orange-950`) |
| **强调色 (Accent)** | `#16A34A` | 生态绿，用于强调成功状态、次要高亮或图表对比数据 (对应 Tailwind `green-600`) |

## 2. 橙色系阶梯 (Orange Palette)
用于构建界面的层次感、边框、悬浮状态等。

*   **浅背景/次要背景**: `#FFEDD5` (orange-100)
*   **边框/分割线**: `#FED7AA` (orange-200)
*   **图表渐变/次要强调**: `#F97316` (orange-500)
*   **深色文本/次要文本**: `#C2410C` (orange-700) / `#9A3412` (orange-800)

## 3. 状态与辅助色 (Status & Utility Colors)

*   **成功/正常 (Success)**: 
    *   背景: `#DCFCE7` (green-100)
    *   图标/文本: `#22C55E` (green-500) / `#16A34A` (green-600)
*   **异常/警告 (Danger/Warning)**: 
    *   背景: `#FEE2E2` (red-100)
    *   图标/文本: `#EF4444` (red-500) / `#B91C1C` (red-700)
*   **信息/数据分类 (Info/Data)**:
    *   背景: `#DBEAFE` (blue-100)
    *   图标/文本: `#3B82F6` (blue-500) / `#2563EB` (blue-600)

## 4. 图表专用色 (ECharts Colors)

在数据可视化大屏和分析图表中，主要使用了以下颜色组合：

*   **雷达图/折线图主色**: `#EA580C` (带透明度填充 `rgba(234, 88, 12, 0.4)`)
*   **柱状图渐变色**: 从 `#F97316` 到 `#FDBA74`
*   **散点图分类色**: 
    *   类别 A (如赣南): `#EA580C`
    *   类别 B (如奉节): `#16A34A`
    *   异常点: `#EF4444`

## 5. 卡片与面板 (Cards & Panels)

*   **毛玻璃面板背景**: `rgba(255, 255, 255, 0.8)` (配合 `backdrop-filter: blur(12px)`)
*   **纯白背景**: `#FFFFFF`

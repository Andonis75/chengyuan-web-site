## 录制视频建议流程

1. 打开网站 → 智能样本分析页
2. 上传「澄迈福橙_CM-001_HSI光谱.csv」→ 点击开始分析
3. 观察AI推理进度（SNV→PCA→SVM→PLS→RF）
4. 查看产地溯源（99.2%置信度）、品质指标、微量成分
5. 切换「双样本对比」模式，上传两个文件对比
